Student Eligibility & Grade Checker

A simple Python program to check student eligibility based on age and marks, assign grades, and give a summary report. Perfect for beginners learning Python basics like input handling, conditionals, and loops.

Features

Enter multiple students’ names, ages, and marks.

Automatically check eligibility:

Must be at least 18 years old.

Must have marks ≥ 50.

Assign grades based on marks:

Top Grade: 90 and above

Grade B: 75–89

Grade C: 50–74

Display a clear summary showing how many students got each grade and how many were not eligible.

How to Use

Clone or download this repository.

Make sure Python 3 is installed.

Run the script in your terminal or IDE:

python student_grades.py


Follow the prompts to input student details.

See the grades for each student and a summary at the end.

Example
Enter number of students: 3

--- Student 1 ---
Enter name: Alice
Enter age: 19
Enter marks: 92
Alice: TOP GRADE

--- Student 2 ---
Enter name: Bob
Enter age: 17
Enter marks: 80
Bob: NOT ELIGIBLE

--- Student 3 ---
Enter name: Charlie
Enter age: 20
Enter marks: 70
Charlie: GRADE C

--- Summary ---
Top Grade students: 1
Grade B students: 0
Grade C students: 1
Not Eligible students: 1
